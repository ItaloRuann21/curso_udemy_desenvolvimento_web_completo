Formatação de textos

HTML contém vários elementos para definir texto com um significado especial.

Elementos de formatação HTML

  • <b> - Texto em negrito
  • <strong> - Texto importante (Parecido com <b>)
  • <i> - Texto em itálico
  • <em> - Texto Enfatizado (Parecido com <i>)
  • <mark> - Texto Marcado (Marcador de texto que já usamos na escola :D)
  • <small> - Texto menor
  • <del> - Texto excluído ( Bom para fazer descontos de preços :D)
  • <ins> - Texto inserido ou com onderline.
  • <sub> - Texto subscrito ( utilizado principalmente em fórmulas químicas H2O)

  • <sup> - Texto Sobescrito ( Pode ser utilizado para notas de rodapé como Italo RuannTM
