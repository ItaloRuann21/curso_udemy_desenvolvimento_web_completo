Citações HTML

Aprendendo uso do: <blockquote>, <q>, <abbr>, <address>, <cite>, <bdo>


<blockquote>

Responsável por fazer citações longas a um determinado texto.
Exemplo:

Aqui é um texto negrito com tag <b>
Aqui é todas as palavras/citações, ou seja, o <blockquote> é usado para fazer citações grandes a um determinado texto. Você pode utilizar ou não um parágrafo para fazer a citação.

<q>

Define uma citação curta.
Os navegadores normalmente inserem aspas ao redor da citação.
Exemplo:

Sobre mim:
Aqui é a utilização da tag <q> para fazer citações curtas. Automaticamente já coloca aspas

<abbr>

Representa uma abreviação/acrônimo e opcionalmente fornece uma descrição para ela.
Muito útil para colocar uma pequena frase e explicar o que é aquela frase.
Exemplo:

Olá pessoal, Aponte o mouse aqui para eu explicar tudo sobre mim.

<address>

Define as informações de contato do autor/proprietário de um documento ou artigo.
As informações de contato podem ser endereço de email, URL, endereço físico, número de telefone, identificador de mídia social e etc.
O texto do <address> é renderizado em itálico e os navegadores sempre adicionam uma quebra de linha antes.
Exemplo:

HTML escrito por Italo Ruann.
Visite-nos em:
example.com.br
Rua Aracaju, Sergipe
BR

<cite>

A tag <cite> define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.)
O texto dentro do escopo <cite>é renderizado em itálico.
Exemplo:

O grito de Edvard Munch. Pintado em 1893.

<bdo>

BDO significa Substituição bidirecional.
A tag HTML <bdo> é usada para substituir a direção atual do texto.
Exemplo:

Aqui o texto está normal. Abaixo, está escrito da direita para a esquerda.
Este texto está escrito da direita para a esquerda.
