Estilos HTML

O atributo style é usado para adicionar estilos css a um elemento como cor, fonte, tamanho e muito mais

O título está usando o atributo style para centralizar ao meio

Podemos definir cores de fundo a páginas utilizando:

  • style="background-color:powderblue;"
o style é o nosso atributo, o background-color é a nossa propriedade e o powderblue é o valor da propriedade e a ; é o sinal utilizado para definir que fechamos o nosso valor.

Para definir cor de fundo de textos, utiliza a mesma sintaxe do style="background-color:powderblue;"
Dessa vez, utiliza a tag e o valor como por exemplo:
h1 style="background-color:powderblue;"

Cor do texto

A propriedade color define a cor de um texto.
Modo de uso: p style="color:blue;"
Onde o style é nosso atributo, color é a propriedade e o blue; é o valor.

Fontes

A propriedade font-family define a fonte a ser usada no HTML
Modo de uso: font-family:verdana;

Tamanho do texto

A propriedade font-size define o tamanho do texto para um elemento HTML.
Modo de uso: font-size:300%;

Alinhamento de Texto

A propriedade text-align define o alinhamento horizontal do texto para um elemento HTML.
Modo de uso: propriedade text-align com valores:

  • center - alinha texto ao centro
  • left - alinha texto a esquerda
  • right - alinha texto a direita.
  • justify - força o texto a ocupar toda a linha da esquerda para a direita.
