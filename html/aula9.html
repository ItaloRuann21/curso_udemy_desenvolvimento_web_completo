Sobre as tabelas

tr - vamos adicionar uma nova linha na nossa tabela. Por exemplo, cada código é uma nova tr

td - é os dados da nossa tr.

th - é o cabeçalho da nossa tabela, ou seja, nosso título

colspan - é um atributo do td que serve para agrupar uma quantidade de colunas

rowspan - é um atributo do td que serve para agrupar a quantidade de linhas.
Exemplo: O desconto ficaria dentro de duas linhas apontando para uma mesma coluna
